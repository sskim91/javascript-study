# 매개변수와 리턴 값

매개변수와 리턴 값을 갖는 함수를 만드는 방법

```javascript
function 함수이름(매개변수...) {
  함수 코드
  return 리턴 값;
}
```

매개 변수에 따라서 여러 결과가 나오는 Javascript의 **Array()** 함수

함수 형태 | 설명 
---------- | ---------- 
Array() | 빈 배열을 만듭니다.
Array(number) | 매개변수 값만큼의 크기를 갖는 배열을 만듭니다.
Array(매개변수..,매개변수) | 매개변수를 배열로 만듭니다.
