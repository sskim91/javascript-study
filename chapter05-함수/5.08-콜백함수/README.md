# 콜백 함수

자바스크립트에서는 함수도 하나의 자료형이다.  
그러므로 함수의 매개변수에 함수를 전달할 수 있다.  
이렇게 매개변수로 전달하는 함수를 **콜백함수**라고 한다.

### 익명 콜백 함수

콜백 함수는 익명 함수로 사용하는 경우가 많다.

```javascript
function callback() {
  코드;
} 
  
callback(function() { //익명 함수로 함수에 매개변수를 전달한다.
  코드;                //매개변수에 익명 함수를 곧바로 입력할 수도 있다.!!
})
```
